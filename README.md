# finallensy

Landing page + prelander for Lensy PRO 2.0.

## Files
```
index.html        ← Main landing page
prelander.html    ← Pre-lander (links to index.html)
css/
  site.css
  vivi.css
assets/
  *.png / *.jpg / *.webp / *.gif / *.svg   ← all referenced images
  site.js
  vivi.js
```

## Push to GitHub

```bash
# 1. Initialize the repo
git init
git add .
git commit -m "initial commit"

# 2. Create the repo on GitHub (do this once at github.com/new → name: finallensy)

# 3. Connect and push
git remote add origin https://github.com/YOUR_USERNAME/finallensy.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.
